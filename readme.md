# WebDriverIO Tests

## Prerequisites
Node.js 18+  
Java (required by Allure)  
Chrome  
Microsoft Edge  

---

## Installation
npm install

---

## Execute Tests
npm test

---

## Tests execute in
Chrome  
Microsoft Edge  

Parallel execution is configured through capabilities.

---

## Generate Allure Report
npm run allure:generate

---

## Open Allure Report
npm run allure:open
